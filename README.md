# Zapier Integration
Timesheet Integration for [Zapier](https://zapier.com). It lets you connect your Timesheet account to a lot of other services.

### Triggers & Actions

#### Actions

Execute one of the following actions:
* Send Report
* Create Team
* Create Project
* Create Task
* Create Rate
* Create Tag

#### Trigger

Get notified when a new item is added:
* New Team
* New Project
* New Task
* New Rate
* New Tag
